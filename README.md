## 🚀 Projects

### [LitLens](https://github.com/trtlbby/litlens) — Literature Review Intelligence PWA
`Next.js` `Python FastAPI` `Gemini AI` `pgvector` `RAG` `Vercel` · *Solo Project · March 2026 – Present*

A Progressive Web Application that helps graduate researchers navigate large collections of academic PDFs during the literature review process.

- **PDF ingestion pipeline** with vector embeddings and Retrieval-Augmented Generation (RAG) for natural-language queries answered with exact in-document citations
- **AI-generated outputs**: thematic clusters, plain-English collection summaries, and gap analysis of under-covered topics
- Integrated **Gemini AI** via API for semantic analysis; deployed on Vercel with 60-second route duration limits for AI-bound endpoints
- Targeting validation by five graduate researchers within a three-month development cycle
